# html + css demo 简介

平常 html 和 css 很多规则，如果不经常使用非常容易忘记
相比逻辑难度，记忆难度更大

# demo 阅读顺序

- base：html+css 的基础知识
- res：各种资源的 html 标签怎么写以及资源怎么引入
- layout：css 常见布局
- background-and-color：背景和颜色等 css
- font：文字字体等相关 css
- margin-and-padding：外边距，线框，内边距使用和区别 css
- align：各种对齐方式 css
- shape：放大，缩小，圆角，旋转，倾斜，阴影，镜像，重叠，平移 等 css 操作
- other: 其他内容或者技巧
